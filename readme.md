# BUAA人工智能学院开题报告与文献综述模板(非官方)

基于github上已有的计算机院开题报告的非官方latex模板修改而来，针对人工智能学院建议使用的word版本(2026年)的开题报告和文献综述进行了适应性调整。

## 主要调整内容
 - 增加了字体文件夹/fonts，处于版权考虑需要自行下载字体(SimHei.ttf & SimSun.ttc & STKaiti.ttf & STXingkai.ttf)然后存放在此文件下，然后打成压缩包后可以直接上传overleaf使用。(overleaf->setting->Compiler->使用XeLatex)
 - 增加了logo文件夹/logo，保存了北航logo的图片文件，供封面使用。
 - buaaAIproposal.cls(主要修改位置)
    - 调整了封面格式，包含北航logo，标题，作者信息的字体与格式。
    - 微调了封面整体排版，尽量与word版本保持一致。
    - 去掉了目录的页眉；页脚编号位置与word版本保持一致。
    - 调整了正文的页眉和页脚；页眉使用word相同的描述和排版，页脚编号位置与word版本保持一致。
    - 参考文献的标题和word版本同步。
 - 调整了各个section的名称(与word模板保持一致)
